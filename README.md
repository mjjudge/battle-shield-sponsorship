# Battle Shield Sponsorship

A standalone WordPress plugin for managing Battle of Evesham shield sponsorships.

The plugin is intended to help administrators manage shield availability, sponsorship payments, sponsor details, artwork uploads, reminders, GDPR requirements, and print-ready A4 sponsorship patches.

## Status

Early development.

## Project Intent

The plugin will support:

- public browsing of available shields
- sponsorship of one or more shields
- Stripe payments
- sponsor logo and text upload
- tokenised post-purchase editing without sponsor logins
- reminder emails for incomplete artwork
- administrator management of shields, sponsors and sponsorships
- print-ready A4 patch generation
- GDPR-aware contact management

## Development

Detailed project files will be populated locally after cloning the repository.
